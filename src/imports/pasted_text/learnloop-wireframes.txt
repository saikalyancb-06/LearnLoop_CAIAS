Design a complete multi-page product wireframe for an AI learning platform called LearnLoop.

The application helps students deeply understand concepts by studying notes, interacting with an AI tutor, using flashcards, and teaching concepts back to the AI using the Feynman learning technique.

This is not a marketing landing page.
Design a full SaaS application interface with multiple screens.

Use a clean modern product layout similar to Notion, Linear, and Miro.

Include:

• Left sidebar navigation
• Top navigation bar
• Main workspace area
• Secondary panels where needed

Use simple wireframe components (boxes, text blocks, icons).

Pages to Design

Create separate wireframes for each page.
